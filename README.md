# equationJS

Simple layout
```
 ┌─────┐           ┌─────┐
 │     │ ┌───────┐ │     │
 │     │ │       │ │     │
 │     │ └───────┘ │     │
 └─────┘           └─────┘
 ```
 
The calculations for this is using a width display, result will be constant:
```
x = (1300 - y) / 2
```
and
```
y = 4x / 3
```
